C++

  • C++ é uma linguagem de programação versátil, poderosa e eficiente, adequada para desenvolver uma variedade de aplicações, desde pequenos programas até grandes sistemas complexos

  • O site oficial é o Standard C++
  • Nome oficial: C++, sendo que, o "++" significa que é um complemento/evolução da linguagem C original

    • Prós

    • Fleixbilidade: O C++ suporta diversos outros recursos e estilos de programação

    • Otimização: O C++ é uma linguagem que permite ao Desenvolvedor que ele melhore/otimize o código, para funcionar melhor nas máquinas

    • Compatibilidade: Todos os Arquivos C e C++ conversão entre si, o que gera uma gigantesca biblioteca de códigos a disposição do programador

    • Contras

    • Complexidade: Ao contrário do HTML, o C++ é uma linguagem extremamente complexa, o que a torna muito mais díficil de entender

    • Aliada a essa complexidade, também há o fator Curva de aprendizado, uma vez que sempre há novos conteúdos sendo adicionados a esta linguagem, fazendo com que o usuário fique sempre tendo que aprender algo novo

    • manutenção manual: O C++, ao contrário de outras linguagens como Java, por exemplo, necessita que a manunteção da memória seja feita de forma manual

      • Projetos que usam C++: Alguns dos projetos são O Outlook, O aplicativo Blender, o Photoshop, a Unreal Engine, o próprio Google Chrome e o Windows
