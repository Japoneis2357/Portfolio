  • Nome oficial: O nome oficial é JavaScript

    • Prós

    • Suporte amplo: Enquanto o HTML e o CSS podem ter incompatibilidade com alguns navegadors, o JavaScript é quase universal, ele é suportado por muitos navegadores

    • Integração com HTML e CSS: O JavaScript pode muito bem ser integrado com HTML e CSS para criar um site cada vez melhor, com relação a UX

    • Fleixbilidade: JavaScript pode ser usado para muitas coisas, sendo uma linguagem que pode ser usada tanto em back-end quanto em front-End

    • Contras

    • Segurança: por conta do JavaScript funcionar no lado do cliente, pode haver quebras na Segurança

    • Desempenho: por ser uma linguagem interpretada e não compilada, pode ser que o computador demore mais para processar o código

    • manutenção complexa: Em sites complexos, pode ser extremamente trabalhoso de revisar e alterar um código nesta linguagem
