HTML

  • html , é o componente básico de todo site ou item web

  • O site oficial é o W3Schools
  • Nome oficial: HyperText Markup Language

    • Prós

    • Simplicidade: é uma linguagem, simples e um tanto quanto fácil de entender e aprender

    • Universalidade: Todos os Sites usam HTML.

    • Compatibilidade: O HTML é uma linguagem que se comunica com outras, o CSS é um bom exemplo, uma vez que é com ele que se estiliza o HTML

    • Contras

    • Limitações: O HTML é muito limitado, uma vez que para que se faça um site mais complexo é necessário acrescentar

    • Diferença entre navegadores: Alguns navegadores "lêem" o HTML de formas diferentes, oque pode gerar conflitos na programação

    • manutenção complexa: uma vez que quanto maior e mais complexo o site, mais trabalhoso será para encontrar o erro

      • Projetos que usam HTML: Alguns dos projetos são O GitHub, o Google, a Netflix, o Twitter, a Amazon e o G1
